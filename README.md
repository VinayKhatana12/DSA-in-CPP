# DSA in C++

My daily Data Structures & Algorithms practice in C++, as part of placement preparation.

## 📌 About
This repo contains solutions to DSA problems solved while preparing for SDE/full-stack roles. Problems are sourced from LeetCode, GeeksforGeeks, and Striver's A2Z Sheet.

## 🗂️ Topics Covered
- Arrays
- Binary Search
- Two Pointers
- Sorting (Merge Sort, Bubble Sort, etc.)
- Linked List
- Strings
- Sliding Window
- Prefix Sum
- HashMaps
